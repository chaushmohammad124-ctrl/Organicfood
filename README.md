# Organic Food - Company Ready React Website

A premium, glossy, animated organic food company website built with React + Vite.

## Features

- Website name: Organic Food
- Home, About, Products, Product Detail, Contact pages
- Video hero section
- Video showcase section
- Premium glossy glassmorphism UI
- Advanced animations with Framer Motion
- Product search and category filter
- Add to Cart button on product cards and detail page
- Responsive cart drawer
- Quantity increase/decrease
- Remove item
- Cart subtotal, delivery, discount, total
- Place Order button
- Company-ready navbar, footer, contact page
- Fully responsive layout

## Run

```bash
npm install
npm run dev
```

## Important

This project uses online video/image URLs for quick preview.
You can replace them with your own files inside:

```txt
public/videos/
public/images/
```

Then update image/video paths in the React files.
